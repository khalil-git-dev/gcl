

La branhe par defaut est DEVELOP (pour tous designeur comme developpeur)


après avoir cloner le projet veuillez suivre ce qui suit:
    
    - Ne jamais tranvailler sur la branche DEVELOP
    - Créer une branche pour chaque fonctionalité (même si c'est seulement un point)
    - Faire la configuration de la base dans le fichier d'environment (.env)
    
    - Après réalisation de la fonctionalitée:
        a) switcher vers DEVELOP
        b) merger votre branche
        c) commiter : Pour les commit la syntaxe a respecter est:
            [ GCEL-Numero_user_story : " Le titre de la tache ou du sous-tache" ]
        d) et enfin push vers la branch DEVELOP


En cas de problème recuperer a nouveau le projet sur le depot GitHub sur la brance DEVELOP
